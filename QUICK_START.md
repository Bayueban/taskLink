## 💡 关键功能速览

### 📊 存储信息
```
使用量: 2.5 MB / 500 MB
进度条显示使用百分比
```

### 📤 导出数据
- **导出所有数据**: 完整备份所有工作区
- **导出当前工作区**: 只导出当前工作区

### 📥 导入数据
- **导入（覆盖）**: 删除现有数据，完全恢复备份
- **导入（合并）**: 保留现有数据，添加新数据

## 🎯 3 个常用场景

### 场景 1: 每周备份
```
1. 打开设置
2. 点击"导出所有数据"
3. 保存到云盘
```
⏱️ 耗时: <5秒

### 场景 2: 换新设备
```
旧设备:
1. 导出所有数据
2. 将文件发送到新设备

新设备:
1. 打开应用
2. 设置 → 导入数据（覆盖）
3. 选择备份文件
```
⏱️ 耗时: <1分钟

### 场景 3: 分享工作区
```
1. 切换到要分享的工作区
2. 设置 → 导出当前工作区
3. 发送文件给对方

对方:
1. 设置 → 导入数据（合并）
2. 选择工作区文件
```
⏱️ 耗时: <30秒

## 🔔 重要提示

### ✅ 建议做的事
1. **立即备份**: 首次使用导出一次数据
2. **定期备份**: 每周或重要变更后备份
3. **多地保存**: 保存在云盘、U盘等多处
4. **测试恢复**: 确保备份文件可用

### ⚠️ 注意事项
1. **覆盖模式**: 会删除所有现有数据，请谨慎
2. **文件安全**: 备份文件包含所有数据，注意保密
3. **格式要求**: 只能导入本应用导出的 JSON 文件
4. **浏览器支持**: 需要现代浏览器（Chrome 90+, Firefox 85+）

## 🆘 遇到问题？

### 问题 1: 找不到设置按钮
**位置**: 左侧面板顶部，"关联空间"标题右侧

### 问题 2: 导出没反应
**解决**: 
1. 检查浏览器是否允许下载
2. 刷新页面重试
3. 查看浏览器控制台错误

### 问题 3: 导入失败
**解决**:
1. 确认文件是 JSON 格式
2. 确认文件由本应用导出
3. 尝试使用最新的备份文件
